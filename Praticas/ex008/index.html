Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> </b> "não semâtica"

Nesta frase temos um termo em destaque usando a tag <strong> </strong> "semântico"

Nesta frase temos um termo em Itálico usando a tag <i> </i> "não semântico"

Nesta frase temos um termo em Ênfase usando a tag <em> </em> "semântico"

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, ultilizamos a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, ultilizamos a tag INS.

Texto Sobrescrito

Para inserir coisas do tipo x9+16, ultilizamos a tag SUP

Texto Subrescrito

Para inserir coisas do tipo H2O, ultilizamos a tag SUB
